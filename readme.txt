== Installation ==
<strong>Using The WordPress Dashboard</strong>

1. Install as regular WordPress plugin.
2. Go your Plugins setting via WordPress Dashboard and activate it.
3. After activating the plugin you will see "Team Ultimate" menu below the comment's menu at the left side on WordPress dashboard and find "Create Shortcode" page to generate team shortcode.<br /> 
<br />

<strong>Using FTP</strong>
1. Extract team-ultimate.zip to your computer.
2. Upload the team-ultimate directory to your wp-content/plugins directory.
3. After activating the plugin you will see "Team Ultimate" menu on the left side of WordPress dashboard and find "Create Shortcode" page to generate team shortcode.

<br />
<strong>How to use short-code?</strong><br />
When plugin active & all setup done please copy shortcode on the shortcode page & paste it anywhere in your post, page or widgets section to display this team showcase.
